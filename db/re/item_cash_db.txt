// Cash Shop Database
// Contains the items sold in the ingame cash shop.
//
// Structure of Database:
// Type,ItemID,Price[,ExpireDuration,Bound]
//
// Type:
//    0: New
//    1: Hot
//    2: Limited
//    3: Rental
//    4: Gear
//    5: Buff
//    6: Heal
//    7: Other
//
// Price:
//    Item cost, in cash points (#CASHPOINTS).
//
// ExpireDuration:
//    Makes the item as rental item after it's bought.
//    Duration is in minute.
//
// Bound:
//    Makes the item as bound item after it's bought.
//    0 - None
//    1 - Account Bound
//    2 - Guild Bound
//    3 - Party Bound
//    4 - Character Bound
